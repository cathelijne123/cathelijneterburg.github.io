PAPERS SECTION

Copy all folders and files into the root of your existing repository.

This package contains:
- an updated _layouts/default.html with a Papers tab
- papers.md, the overview page
- seven individual pages in papers/
- seven placeholder preview figures
- an updated assets/css/style.css

Replace the placeholder titles, descriptions, links, authors, and figures.

To replace a preview figure, either:
1. overwrite the matching SVG file, or
2. change the image filename in papers.md and the individual paper page.

Then run:
git add .
git commit -m "Add papers section"
git push
